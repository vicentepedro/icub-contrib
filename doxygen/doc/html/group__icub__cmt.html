A tracker based on the CMT algorithm.

Description

This is a yarp wrapper of the CMT algorithm (a.k.a. Consensus-based Matching and Tracking) for tracking objects.
The C++ version of the method is taken from the libCMT library.

Available RPC services are:

(notation: [.] identifies a vocab, <.> specifies a double, "." specifies a string)

INIT
action: init CMT with the template given by the bounding box.
format: [init] <top-left.x> <top-left.y> <bottom-right.x> <bottom-right.y>

START
format: [start]
action: start tracking.

STOP
format: [stop]
action: stop tracking.

Libraries

  • YARP libraries.
  • OpenCV (with C++ layer).

Ports Created

  • / <modName>/img:i receives the image acquired from the camera.
  • / <modName>/img:o streams out the image containing tracked object.
  • / <modName>/data:o streams out information about the tracked object in the following format: <top-left.x> <top-left.y> <top-right.x> <top-right.y> <bottom-right.x> <bottom-right.y> <bottom-left.x> <bottom-left.y>.
  • / <modName>/rpc processes rpc requests.

Parameters

–name name

  • specify the module stem-name, which is cmt by default. The stem-name is used as prefix for all open ports.

Tested OS

Linux, Windows
